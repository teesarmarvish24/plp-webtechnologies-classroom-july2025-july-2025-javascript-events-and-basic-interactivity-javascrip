What is the DOM?
The Document Object Model. A tree-like interface that lets JavaScript interact with HTML elements.
What is an 'Event Listener'?
A function that "listens" for a specific event (like a 'click' or 'hover') on an element and then runs a block of code.

CSS Concepts

CSS (Cascading Style Sheets) is what makes the web look good. It handles colors, fonts, layout, and animations.

For example, the flexbox and grid properties are powerful tools for aligning content on a page. This tab layout itself is controlled by CSS!

Effective Study Methods

  • Active Recall: Actively trying to retrieve information from memory, just like you did with the flashcards.
  • Spaced Repetition: Reviewing information at increasing intervals over time to improve long-term retention.
  • The Pomodoro Technique: Work in focused 25-minute intervals (called "Pomodoros") separated by short 5-minute breaks.
  • Feynman Technique: Try to explain a complex concept in simple terms, as if you were teaching it to someone else.
